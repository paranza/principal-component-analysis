# Principal Component Analysis

This repository contains a group project developed as part of an Introduction to Statistics course.

The objective of the project was to apply PCA to a real dataset, interpret the principal components, and analyse how variance is distributed across dimensions.

## Project Overview

The analysis includes:

- Standardisation of variables  
- Computation of the covariance/correlation matrix  
- Eigenvalue and eigenvector decomposition  
- Variance explained by principal components  
- Interpretation of loadings  
- Graphical visualisation of projections  

The project emphasises both the mathematical foundations of PCA and its practical use as a tool for data compression and structure detection.
